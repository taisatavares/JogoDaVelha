Projeto Jogo da Velha desenvolvido em Java. 
A intenção do jogo é que o jogador enfrente o computador e posteriormente verifique o resultado.
Nessa disputa temos 3 opções:

   1 - Jogador ganha
   2 - Computador ganha
   3 - Empate
